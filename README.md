# GBL Cash Flow Consolidation Dashboard

Single-file HTML tool for consolidating monthly cash flow across 21 GBL portfolio brands.

## Usage
Open `index.html` directly in Chrome — no server needed.

## Live Tool
Self-hosted on EC2 (nginx): https://gbl-cff.duckdns.org/

Dev preview (snapshot of `dev` branch): https://gbl-cff.duckdns.org/dev/

## Monthly Workflow
1. Upload Jun-26+ brand files
2. Map groupings by brand
3. Recon vs brand summary
4. IC Loans check
5. Other Line Items check
6. Confirm & Bake In

## Historical Data
FY22–May26 pre-loaded. Each month's data is baked in after confirmation.
